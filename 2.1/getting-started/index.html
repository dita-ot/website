Getting Started with the DITA Open Toolkit

The Getting Started Guide is designed to provide a guided exploration of the DITA Open Toolkit. It is geared for an audience that has little or no knowledge of build scripts or DITA-OT parameters. It walks the novice user through installing the client version of the toolkit and running a prompted build.

  1. Installing the client package
    For the simplest installation experience, install the client package. This package can be installed on Linux, Mac OS X, and Windows. It contains everything that you need to run the DITA-OT except for Java.
  2. Building output using the dita command
    You can invoke DITA-OT and build output using the dita command.
